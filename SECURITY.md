# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, report it privately by email:

- support@platfone.com

Please include:

- A clear description of the issue
- Steps to reproduce the problem
- Expected impact
- Any relevant logs, screenshots, or proof of concept

Do not open public issues for security vulnerabilities.

## Response Process

We will review reports and determine the appropriate remediation steps based on severity and impact. Compensation, if any, is handled at our discretion.
