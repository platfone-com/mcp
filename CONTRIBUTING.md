# Contributing

Thanks for your interest in contributing to the Platfone MCP server!

## Getting Started

1. Fork the repository
2. Clone your fork and install dependencies:
   ```bash
   git clone https://github.com/<your-username>/mcp.git && cd mcp
   npm install
   ```
3. Create a branch for your change:
   ```bash
   git checkout -b my-change
   ```
4. Follow the [Development Guide](docs/DEVELOPMENT.md) for setup and testing.

## Submitting Changes

- Open a pull request against `main`.
- Describe what changed and why.
- Keep PRs focused — one concern per PR.

## Reporting Issues

Open an issue at [github.com/platfone-com/mcp/issues](https://github.com/platfone-com/mcp/issues).

For security vulnerabilities, see [SECURITY.md](SECURITY.md).

## Code Style

This project uses [Prettier](https://prettier.io/) for formatting. Run `npm run format` before committing.
